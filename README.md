# 机器学习学习记录

## 项目概述
本仓库包含两个机器学习学习路径的实践记录：
1. **基于 Tom M. Mitchell《机器学习》教材**：系统学习经典机器学习理论，从最基础的定义出发，逐步实现经典算法
2. **Andrew's 吴恩达教授视频学习**：记录观看吴恩达教授机器学习视频课程的学习过程和实践

核心目标是把理论概念落地为可运行的代码，吃透机器学习的核心逻辑，拒绝黑盒式的浅层应用。

## 已完成内容

### 1. 基于 Tom M. Mitchell《机器学习》教材
- **教材第一章 引言**：完整核心案例实践
  - 机器学习核心定义拆解与映射
  - 手写数字识别（MNIST数据集）完整实现
  - 西洋跳棋自学习博弈程序完整实现

### 2. Andrew's 吴恩达教授视频学习
- **视频课程学习记录**：系统观看吴恩达教授的机器学习视频课程
- **实践项目**：基于视频内容的代码实现和实验
- **学习笔记**：关键概念和算法的总结与理解

## 核心实现特点
1. **理论与实践一一对应**：所有代码实现均严格对应理论概念
2. **完整的机器学习闭环**：从数据获取、模型训练到效果评估的完整流程
3. **详细的学习笔记**：记录每个实现的思路、遇到的问题和解决方案
4. **可复现性**：提供完整的代码和环境配置，确保实验结果可复现
